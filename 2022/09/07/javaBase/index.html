整数类型表示范围

  • java各整数类型有固定的表数范围和字段长度,不受具体OS的影响,以保证java程序的可移植性.
  • java的整型常量默认为int型,声明long型常量需后加”l”或”L”
  • java程序中变量通常声明为int型,除非不足以表示较大的数,才使用long

基本数据类型之间的运算

前提: 这里讨论只是7种基本数据类型变量间运算,不包含boolean类型

1. 自动类型提升

当容量小的数据类型的变量与容量大的数据类型的变量做运算时,结果自动提升为容量大的数据类型

此时的容量大小指的是表示数范围的大和小,比如float容量要大于long的容量

特殊的：当byte,char,short三种类型做运算时,结果为int型

byte、char、short –> int –> long –> float –> double –> String

2. 强制类型转换

自动类型提升运算的逆运算。

public class test{
    public static void main(String[] args){
        double i = 1.1
        // int j = i //报错:不兼容类型:从double转到int
        int j = (int)i //需要强制类型转换,但会造成精度丢失
    }
}

计算机中不同进制的使用说明

  • 二进制(binary):0,1 满2进1,以0b或0B开头
  • 十进制(decimal) 0-9, 满10进1
  • 八进制(octal):0-7, 满8进1,以数字0开头
  • 十六进制(hex): 0-9及A-F, 满16进1,以0x或0X开头

变量之进制

  • java整数常量默认int类型,当用二进制定义整数时,其第32位是符号位;当long类型时,二进制默认占64位,第64位是符号位
  • 二进制的整数有如下三种形式

原码:直接将一个数值换成二进制数,最高位是符号位.

负数的反码: 是对原码按位取反,只是最高位(符号位)确定为1.

负数的补码: 其反码加1.

  • 计算机二进制补码的形式保存所有的整数.

    正数的原码,反码,补码都相同

负数的补码是其反码+1

算数运算符使用

  • (前)++:先自增1,后运算
  • (后)++:先运算,后自增1
    class test {
    public static void main(){
        int a1 = 10;
		int a2 = ++a1;
		System.out.println("a1 =" + a1 + ",a2 ="+  a2); //a1=11 , a2=11
		int a3 = 21;
		int a4 = a3++;
		System.out.println("a3 = "+a3+",a4 = "+a4); //a3=22 , a4=21
    }
}

class test {
    public static void main(String[] args){
        int num = 113;
        int bai = num / 100; //因为是百位数,当除100时最终类型为int型,所以值等于多少,百位就是多少
        int shi = num % 100 / 10; //当num除100取余时,取的余就是十位数(2位),然后取余除10得到的就是十位数
        int ge = num % 10; //因为最终得到的值是整型,所以除以10最终的余数就是个位数
        System.out.println("百位为:"+bai);
        System.out.println("十位为:"+shi);
        System.out.println("个位为:"+ge);
    }
}

&与&&区分:

相同点1. &与&&的运算结果相同

相同点2. 当符号左边是true时,二者都会执行符号右边的运算

不同点: 当符号左边为false时,&继续执行符号右边的运算,&&不再执行符号右边的运算

|与||区分:

相同点1. |与||的运算结果相同

相同点2. 当符号左边是false时,二者都会执行符号右边的运算

不同点: 当符号左边为true时,|继续执行符号右边的运算,||不再执行符号右边的运算

运算符: 位运算符
运算符 运算 规范

>> 右移 3>>1=1 –> 3 / 2 = 1
>>> 无符号右移 3>>>1=1 –> 3 / 2=1

  • 左移n位就是乘以2的n次方。

其功能把”<<”左边的运算数的各二进位全部左移若干位，由”<<”右边的数指定移动的位数，高位丢
弃，低位补0。

(1) 例如3这个数值，存储在1个字节的空间中，其二进制格式是0000 0011，如果我们把这个数值往左移动2位，那么，就得到0000 1100，此时，表示的十进制是12。

(2) 例如2这个数值，存储在1个字节的空间中，其二进制格式是0000 0010，如果我们把它往左移动3位，就得到0001 0000，此时，十进制表示16
。

那么，我们可以总结：

(1) 数值3往左移动2位，变成12数值；如同3 * 2^2 = 3 * 4 = 12；

(2) 数值2往左移动3位，变成16数值；如同 2*2^3 = 2 * 8 = 16；

那么，我们可以归纳总结，数值N往左移动X位，得到是数值是N*2^X，就是数值N乘以2的X次方。注意：我们这种计算方式，是在不考虑“溢出”的情况下。

  • 右移n位就是除以2的n次方。

其功能是把”>>”左边的运算数的各二进位全部右移若干位，”>>”右边的数指定移动的位数。

&运算

按位与”&”功能是参与运算的两数各对应的二进位相与。只有对应的两个二进位均为1时，结果位才为1 ，否则为0。参与运算的数以补码方式出现。

10的补码(00001010),6的补码(00000111)

10: 00001010

6: 00000111
每位对应&运算得 00000010(对应十进制2)
所以10&6结果是2
